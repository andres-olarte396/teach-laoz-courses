# Sistema de Evaluaciones

## 📊 Visión General

Este directorio contiene todos los recursos de evaluación para el Curso de IA y Automatización, organizados por tipo y fase del curso.

## 📁 Estructura de Directorios

### 📝 [quizzes/](./quizzes/)

Contiene todos los cuestionarios por fase:

- Preguntas de selección múltiple
- Preguntas verdadero/falso
- Preguntas de desarrollo corto
- Auto-evaluaciones interactivas

### 📋 [rubrics/](./rubrics/)

Matrices de evaluación detalladas:

- Criterios de calificación por proyecto
- Escalas de competencia
- Indicadores de desempeño
- Retroalimentación estructurada

### 🎯 [projects/](./projects/)

Especificaciones de proyectos evaluativos:

- Proyectos integradores por fase
- Casos de estudio reales
- Rúbricas específicas de proyecto
- Ejemplos de entregables

## 🎯 Metodología de Evaluación

### Evaluación Formativa (70%)

- **Quizzes semanales:** 20%
- **Ejercicios prácticos:** 30%
- **Proyectos menores:** 20%

### Evaluación Sumativa (30%)

- **Examen teórico por módulo:** 15%
- **Proyecto integrador final:** 15%

## 📈 Criterios de Calificación

| Nivel | Rango | Descripción |
|-------|-------|-------------|
| **Excelente** | 90-100% | Supera expectativas, demuestra maestría |
| **Sobresaliente** | 80-89% | Cumple todos los objetivos con calidad |
| **Satisfactorio** | 70-79% | Cumple objetivos básicos |
| **Necesita Mejora** | 60-69% | Cumple parcialmente objetivos |
| **Insuficiente** | <60% | No cumple objetivos mínimos |

## ⚡ Evaluación Automatizada

### Sistema de Auto-Corrección

- Scripts de validación para código Python
- Verificación automática de outputs
- Feedback inmediato en ejercicios
- Dashboard de progreso individual

### Herramientas de Evaluación

```python
# Ejemplo de auto-evaluación
from evaluation_tools import CodeValidator, QuizChecker

validator = CodeValidator()

```

## 🔄 Proceso de Retroalimentación

1. **Evaluación Automática** (inmediata)
2. **Revisión Manual** (48-72 horas)
3. **Retroalimentación Detallada** (criterios específicos)
4. **Oportunidad de Mejora** (reenvío permitido)

## 📊 Seguimiento de Progreso

### Métricas de Desempeño

- Puntaje promedio por fase
- Tiempo de completión de actividades
- Nivel de participación en foros
- Calidad de entregas de proyectos

### Reportes de Progreso

- Dashboard individual del estudiante
- Reporte semanal de avance
- Alertas de riesgo académico
- Certificación de competencias

## 🎓 Certificación

### Requisitos para Certificación

- **Nota mínima general:** 75%
- **Completar todas las fases:** 100%
- **Proyecto final aprobado:** Sí
- **Participación en comunidad:** Activa

### Tipos de Certificación

1. **Certificado de Finalización** - Completa el curso con nota mínima
2. **Certificado de Excelencia** - Completa con nota >90%
3. **Badge de Especialización** - Demuestra expertise en área específica
