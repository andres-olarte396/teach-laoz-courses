# Fase 03: Machine Learning

## 🎯 Objetivos de Aprendizaje

Al finalizar esta fase, el estudiante será capaz de:

- Comprender los algoritmos fundamentales de Machine Learning
- Implementar modelos de clasificación y regresión
- Aplicar técnicas de validación y evaluación de modelos
- Utilizar scikit-learn para desarrollo de modelos
- Optimizar hiperparámetros y mejorar rendimiento

## ⏱️ Duración Estimada

**Tiempo total:** 20-25 horas

- Teoría: 12 horas
- Práctica: 10-13 horas
- Evaluación: 2 horas

## 🔧 Prerequisitos

- Completar Fase 02: Análisis de Datos
- Conocimientos sólidos de estadística
- Experiencia con Pandas y NumPy

## 📊 Evaluación

- **Implementación de algoritmos:** 35% (código funcional)
- **Proyecto de clasificación:** 25% (caso completo)
- **Proyecto de regresión:** 25% (predicción numérica)
- **Evaluación teórica:** 15% (conceptos clave)
- **Nota mínima:** 75%
