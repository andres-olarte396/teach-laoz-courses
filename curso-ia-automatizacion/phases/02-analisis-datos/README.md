# Fase 02: Análisis de Datos

## 🎯 Objetivos de Aprendizaje

Al finalizar esta fase, el estudiante será capaz de:

- Dominar las librerías fundamentales de análisis de datos (Pandas, NumPy)
- Realizar exploración y limpieza de conjuntos de datos
- Crear visualizaciones efectivas con Matplotlib y Seaborn
- Aplicar estadística descriptiva e inferencial
- Preparar datos para modelos de Machine Learning
