# 📚 Ejemplos Incrementales - Fase 00: Fundamentos de Programación

Este directorio contiene una serie de ejemplos progresivos que te llevarán desde los conceptos más básicos de Python hasta la creación de proyectos completos. Cada ejemplo construye sobre el conocimiento adquirido en los anteriores.

## 🎯 Metodología de Aprendizaje

Los ejemplos están diseñados con una **progresión incremental**, donde cada archivo:

✅ **Introduce UN concepto nuevo principal**  
✅ **Refuerza conceptos anteriores**  
✅ **Incluye ejercicios prácticos**  
✅ **Prepara para el siguiente nivel**  

## 📋 Secuencia de Ejemplos

### 🌱 **Nivel Principiante**

#### 📁 `ejemplo_01_hola_mundo.py`
- **Concepto**: Primeros pasos en Python
- **Aprenderás**: `print()`, variables básicas, comentarios
- **Tiempo estimado**: 15 minutos
- **Prerrequisitos**: Ninguno

#### 📁 `ejemplo_02_variables_tipos.py`
- **Concepto**: Tipos de datos fundamentales
- **Aprenderás**: `int`, `float`, `str`, `bool`, `type()`
- **Tiempo estimado**: 20 minutos
- **Prerrequisitos**: ejemplo_01

#### 📁 `ejemplo_03_operaciones_basicas.py`
- **Concepto**: Operadores y cálculos
- **Aprenderás**: Aritméticos, comparación, calculadora de compras
- **Tiempo estimado**: 25 minutos
- **Prerrequisitos**: ejemplo_02

#### 📁 `ejemplo_04_entrada_datos.py`
- **Concepto**: Interacción con usuarios
- **Aprenderás**: `input()`, conversión de tipos, validación básica
- **Tiempo estimado**: 30 minutos
- **Prerrequisitos**: ejemplo_03

### 🌿 **Nivel Principiante-Intermedio**

#### 📁 `ejemplo_05_condicionales.py`
- **Concepto**: Lógica de decisiones
- **Aprenderás**: `if/elif/else`, operadores lógicos, sistema de calificaciones
- **Tiempo estimado**: 35 minutos
- **Prerrequisitos**: ejemplo_04

#### 📁 `ejemplo_06_bucles.py`
- **Concepto**: Repetición y automatización
- **Aprenderás**: `while`, `for`, `range()`, análisis de texto interactivo
- **Tiempo estimado**: 45 minutos
- **Prerrequisitos**: ejemplo_05

### 🌳 **Nivel Intermedio**

#### 📁 `ejemplo_07_listas.py`
- **Concepto**: Colecciones ordenadas
- **Aprenderás**: Listas, métodos, gestión de tareas completa
- **Tiempo estimado**: 50 minutos
- **Prerrequisitos**: ejemplo_06

#### 📁 `ejemplo_08_diccionarios.py`
- **Concepto**: Datos estructurados clave-valor
- **Aprenderás**: Diccionarios, inventario, análisis por categorías
- **Tiempo estimado**: 60 minutos
- **Prerrequisitos**: ejemplo_07

### 🌲 **Nivel Intermedio-Avanzado**

#### 📁 `ejemplo_09_funciones.py`
- **Concepto**: Código modular y reutilizable
- **Aprenderás**: Definición de funciones, parámetros, calculadora científica
- **Tiempo estimado**: 70 minutos
- **Prerrequisitos**: ejemplo_08

#### 📁 `ejemplo_10_modulos_proyecto.py`
- **Concepto**: Organización avanzada y librerías
- **Aprenderás**: Módulos, clases, archivos, agenda personal completa
- **Tiempo estimado**: 90 minutos
- **Prerrequisitos**: ejemplo_09

## 🎓 Cómo usar estos ejemplos

### 📖 **Para cada ejemplo:**

1. **📚 Lee el código completo** primero para entender la estructura
2. **💻 Ejecuta el programa** y prueba todas las funcionalidades
3. **🔧 Modifica parámetros** para ver cómo cambia el comportamiento
4. **✏️ Completa el ejercicio** propuesto al final
5. **🤔 Reflexiona** sobre cómo se conecta con el ejemplo anterior

### 🏃‍♂️ **Ritmo recomendado:**

- **Principiantes**: 1-2 ejemplos por día
- **Con experiencia**: 2-3 ejemplos por día
- **Avanzados**: Todos en una sesión intensiva
