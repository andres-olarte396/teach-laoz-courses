# Fase 04: RPA Básico

## 🎯 Objetivos de Aprendizaje

Al finalizar esta fase, el estudiante será capaz de:

- Comprender los principios fundamentales de RPA
- Configurar y utilizar herramientas RPA populares
- Automatizar tareas repetitivas en interfaces gráficas
- Desarrollar bots básicos para procesos empresariales
- Integrar RPA con Python y otras tecnologías
