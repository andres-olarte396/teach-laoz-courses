# Fase 05: APIs de IA

## 🎯 Objetivos de Aprendizaje

Al finalizar esta fase, el estudiante será capaz de:

- Integrar APIs de IA en aplicaciones y automatizaciones
- Utilizar servicios de OpenAI, Google Cloud AI, y Azure Cognitive Services
- Desarrollar aplicaciones con procesamiento de lenguaje natural
- Implementar sistemas de visión computacional
- Crear chatbots y asistentes inteligentes

## ⏱️ Duración Estimada

**Tiempo total:** 20-24 horas

- Teoría: 8 horas
- Práctica: 14-16 horas
- Evaluación: 2 horas

## 🔧 Prerequisitos

- Completar Fase 03: Machine Learning
- Conocimientos de desarrollo web básico
- Cuentas en proveedores de IA (OpenAI, Google Cloud, etc.)

## 📊 Evaluación

- **Chatbot inteligente:** 35% (OpenAI API + funcionalidades)
- **Sistema de visión:** 25% (procesamiento de imágenes)
- **Análisis de texto:** 25% (NLP y sentiment analysis)
- **Integración completa:** 15% (aplicación multi-API)
- **Nota mínima:** 75%
